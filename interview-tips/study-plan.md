# Study Plan

## 12-Week Interview Preparation Plan

### Phase 1: Foundations (Weeks 1–4)

**Goal:** Solidify core data structures and basic algorithms.

| Week | Focus                          | LeetCode Problems (Easy/Medium) |
|------|-------------------------------|----------------------------------|
| 1    | Arrays, Strings, Hash Tables   | Two Sum, Valid Anagram, Group Anagrams, Longest Substring |
| 2    | Linked Lists, Stacks, Queues   | Reverse Linked List, Valid Parentheses, Min Stack |
| 3    | Trees & BSTs                   | Max Depth, Level Order Traversal, Validate BST |
| 4    | Recursion & Binary Search      | Climbing Stairs, Binary Search, Merge Sort |

---

### Phase 2: Core Algorithms (Weeks 5–8)

**Goal:** Master algorithmic patterns.

| Week | Focus                          | LeetCode Problems (Medium)       |
|------|-------------------------------|----------------------------------|
| 5    | BFS, DFS, Graphs               | Number of Islands, Course Schedule, Clone Graph |
| 6    | Dynamic Programming (1D)       | Coin Change, House Robber, Longest Increasing Subsequence |
| 7    | Dynamic Programming (2D)       | Unique Paths, LCS, Edit Distance |
| 8    | Heaps, Tries, Union-Find       | Top K Frequent, Implement Trie, Redundant Connection |

---

### Phase 3: Advanced & System Design (Weeks 9–12)

**Goal:** Hard problems + system design fundamentals.

| Week | Focus                          | Activity                         |
|------|-------------------------------|----------------------------------|
| 9    | Advanced DP, Backtracking      | Burst Balloons, N-Queens, Word Search II |
| 10   | System Design Basics           | Study: URL Shortener, Rate Limiter, Key-Value Store |
| 11   | System Design Advanced         | Study: Twitter Feed, Chat System, Video Streaming |
| 12   | Mock Interviews & Review       | Full mock interviews, review weak areas |

---

## Daily Practice Routine

| Time     | Activity                                      |
|---------|-----------------------------------------------|
| 30 min  | Review 1 concept (algorithm/data structure)   |
| 60 min  | Solve 2-3 LeetCode problems                   |
| 20 min  | Review solutions and read editorial           |
