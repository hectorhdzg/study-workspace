## Why GenAI in Tech Interviews?

Companies increasingly ask about GenAI in system design and coding rounds:
- **System design:** "Design a customer support chatbot with RAG"
- **Coding:** "Build a semantic search pipeline"
- **Architecture:** "How would you add AI features to an existing product?"
- **Trade-offs:** Cost vs latency vs quality, when to use AI vs traditional approaches
