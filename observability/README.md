## Why Observability in Tech Interviews?

Observability is critical for operating production systems at scale:
- **System design rounds:** "How would you monitor this system?" is a common follow-up
- **Incident management:** Debugging production issues is a daily reality
- **SRE / Platform roles:** Core domain knowledge
- **Trade-offs:** Cost vs visibility, sampling rates, cardinality, storage retention
