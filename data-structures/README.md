## Choosing the Right Data Structure

| Use Case                        | Best Structure                    |
|---------------------------------|-----------------------------------|
| Fast random access              | Array                             |
| Fast insert/delete at front/back| Doubly Linked List, Deque         |
| LIFO order                      | Stack                             |
| FIFO order                      | Queue                             |
| Min/Max element fast            | Heap (Priority Queue)             |
| Fast key-value lookup           | Hash Map                          |
| Sorted key-value lookup         | BST / Sorted Map                  |
| Prefix string search            | Trie                              |
| Relationship between entities   | Graph                             |
