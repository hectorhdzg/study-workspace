## Choosing the Right Algorithm

| Problem Type               | Consider                             |
|---------------------------|--------------------------------------|
| Find shortest path         | BFS, Dijkstra                        |
| Find any path              | DFS, BFS                             |
| Optimize a value           | Dynamic Programming, Greedy          |
| Process elements in order  | Sorting + Binary Search              |
| Substring/subarray         | Sliding Window, Two Pointers         |
| Combinations/permutations  | Backtracking                         |
| Fast lookups / deduplication | Hash Map / Hash Set                |
